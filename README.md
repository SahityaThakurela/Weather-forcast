### Weather Forecast

Weather Forecast is a web-based application that allows users to check real-time weather conditions for any city around the globe. By leveraging a RESTful Weather API, the application fetches accurate, up-to-date meteorological data and presents it in a clean, intuitive interface.

The project is built entirely with vanilla web technologies, demonstrating a strong understanding of DOM manipulation, handling asynchronous JavaScript (Promises/Fetch API), and responsive design without relying on heavy frontend frameworks.

Key Features:

**Real-Time Search**: Users can input a city name to instantly fetch and view current weather conditions.

**Detailed Metrics**: Displays crucial weather data including temperature, humidity levels, wind speed, and dynamic weather condition icons.

**Dynamic UI**: The interface adapts smoothly to the data provided, offering an easy-to-read layout that updates without requiring page reloads.

**Responsive Design**: Styled to provide a smooth and visually appealing experience across both desktop and mobile devices.

Tech Stack:

**Frontend**: HTML5, CSS3, JavaScript (Vanilla ES6+)

**API**: RESTful Weather API (e.g., OpenWeatherMap / WeatherAPI)

**Deployment**: Hosted live on Vercel
